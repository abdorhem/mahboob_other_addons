Odoo Recently Viewed Records

It is Quite easy to lose track of the task at hand when engaging in multiple activities on a whole ERP system simultaneously. A simple solution is Odoo Recently Viewed Records Module. It saves the recent activity of the user in the Odoo backend and displays it to help the user see the last viewed records arranged according to time in descending order. The recently viewed history is also available in the user's profile under settings in the back-end.

Salient Features
  • Displays user's recent viewed record history.
  • Records list saved in users data.
